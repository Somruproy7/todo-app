SDE-1 (MERN Stack) Assignment 1. Product Overview The To-Do List App allows users to create, edit, delete, and search for tasks. Tasks can be scheduled with a specific date and time, and users can easily track their progress through the home screen interface. The app will organize tasks by week and visually display the number of open and closed tasks for each week. 2. Objective Provide users with an intuitive and efficient way to manage their tasks. Ensure that users can easily create, update, and track their to-do items while receiving visual feedback on their weekly progress. 3. Key Features 3.1 Task Management • Create Task: o Users can create tasks with the following attributes: ▪ Title (mandatory) ▪ Description (optional) ▪ Date & Time (mandatory) ▪ Priority (Low, Medium, High) (optional) • Edit Task: o Users can update any of the task attributes (title, description, date & time, priority). • Delete Task: o Users can delete tasks from their list. • Search Task: o Users can search for tasks using keywords (e.g., title, description). o Search results should display relevant tasks with options to view, edit, or delete. 3.2 Home Screen Overview • Weekly Organization: o Tasks will be grouped and displayed by week. o Weeks will start on Monday and end on Sunday. • Task Summary Cards: o Each week's card will show: ▪ Number of Open Tasks ▪ Number of Completed Tasks o Clicking on a week’s card will expand it to show all tasks for that week. 3.3 Task Status Update • Users can mark tasks as Completed or In Progress. • Once a task is marked as completed, it will update the week's card to reflect the number of closed tasks. 4. User Flow 1. Home Screen: o Displays weekly cards summarizing tasks (open vs. completed). o Tap on a week to view detailed tasks for that week. 2. Create Task: o Tap on the "Add Task" button. o Enter task details (title, description, date & time, priority). o Tap "Save" to add the task to the selected week. 3. Edit Task: o Select a task from the week’s list. o Edit task details and save changes. 4. Delete Task: o Swipe left on a task in the week’s list. o Tap “Delete” to remove it. 5. Search Task: o Tap on the search icon. o Enter keywords to find specific tasks. o View, edit, or delete tasks from the search results.
